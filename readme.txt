# libmatematica

A `libmatematica` é uma biblioteca em C para cálculos matemáticos básicos, incluindo funções para calcular o seno, cosseno e tangente de um ângulo em radianos.

## Instalação

Para usar a `libmatematica`, clone este repositório usando o Git:

```bash
git clone https://github.com/seu-usuario/libmatematica.git
